OP Mainnet is a public blockchain network built on the OP Stack that runs Ethereum-compatible transactions at significantly lower cost and higher throughput than Ethereum mainnet, settling its transaction data back to Ethereum for security. Every application that works on Ethereum works on OP Mainnet with no code changes, making it a drop-in scaling solution for the Ethereum ecosystem.
